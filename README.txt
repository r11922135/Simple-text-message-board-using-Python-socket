phase 2實作了以下功能：
1. 註冊、登入、登出功能(有實作cookie)
2. 留言板功能，若有更新內容會即時顯示在頁面
3. 影片、聲音檔撥放功能
4. multithread功能，每當有新的request就會開啟一個新的thread去處理
5. https(使用ssl實作)
6. 美觀的使用者介面
7. 登入或註冊的頁面有實作input檢查功能，例如輸入空白或試圖登入不存在的帳號皆會跳出警告
8. 在尚未登入的情況下試圖進入留言板頁面會強制跳回登入頁面

說明：
1. 直接在localhost執行server.py
2. 於瀏覽器網址列輸入https://localhost:11135/login.html進入網頁
3. 如果瀏覽器跳出安全性警告可直接略過
4. 可註冊新帳號或登入現有帳號
5. 登入後即可留言or瀏覽影片檔、聲音檔
6. 本次作業內容使用chatgpt 4協助完成
